# Derived from mechabar theme for Waybar

Credit to [mechbar](https://github.com/sejjy/mechabar) for this great-looking theme. 

Customized a bit to fit my setup and preference.

## Modifications

- Moved powermenu module to middle and removed Arch logo
- Moved audio module to right-most position and omitted some of the stylings
- Adwaita Mono + Iosevka combo for compact media and program titles
- Colored workspace rhombi
- Click left-most menu item to start wofi
